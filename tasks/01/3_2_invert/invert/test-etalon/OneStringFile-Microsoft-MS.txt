MS Visual C++ (MSVC) — компилятор для приложений на языке C++, разработанный корпорацией MS и поставляемый либо как часть комплекта MS Visual Studio, либо отдельно в виде бесплатного функционально ограниченного комплекта MS Visual Studio Community Edition (ранее Visual C++ Express Edition). Сменил интегрированную среду разработки MS QuickC. Поддерживает стандарт C11, C++23. Предшественник Visual C++ назывался MS C/C++. Также были MS QuickC 2.5 и MS QuickC для Windows 1.0. Компилятор Visual C++ по-прежнему известен как MS C/C++ и на момент выпуска Visual C++ 2015 Update 2 имеет версию 14.0.23918.0.
